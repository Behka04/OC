# OC
Генерация данных: Сначала создается большой вектор случайных чисел.
Однопоточный расчет: Затем вычисляется сумма всех элементов в векторе с использованием одного потока, и измеряется время, затраченное на это.
Многопоточный расчет:
Программа определяет, сколько ядер CPU доступно.
Вектор делится на части, по одной части на каждый поток.
Каждый поток запускается и параллельно вычисляет сумму своей части вектора.
Основной поток ждет, пока все дочерние потоки завершатся (join).
Затем основной поток суммирует все частичные результаты, чтобы получить общую сумму.
Измеряется время, затраченное на многопоточный расчет.
Сравнение: В конце программа выводит обе суммы (они должны быть очень близки или одинаковы, за исключением возможных небольших ошибок округления из-за арифметики с плавающей точкой) и время выполнения, демонстрируя потенциальное ускорение при использовании нескольких потоков.
Ожидаемый результат: На системах с несколькими ядрами многопоточная версия должна работать значительно быстрее, чем однопоточная, поскольку работа распределяется между несколькими ядрами, которые выполняют вычисления параллельно.
